How I work

AI-first by default

Every project is shaped around the assumption that an LLM is part of the toolchain — not as a gimmick, but as a force multiplier for analysis, automation, and scaffolding. Claude Code, Claude API, and local models share the same goal: collapse the time from question to shippable artifact.

Local compute, no excuses

Mac Studio M3 Ultra · 256 GB unified memory means 70B+ dense models, 100B+ MoE quantized, and full datasets in memory — no cloud round-trips. Fine-tuning, distillation, and eval harnesses run end-to-end on the desk.

Reproducible & visible

Kaggle leaderboards, public GitHub repos, and clean READMEs over private one-offs. If it doesn't have a benchmark, a notebook, or a write-up, it didn't ship.

Domain depth

Online travel, intelligent automation, and personal quant research — the same pattern each time: messy data, real-money stakes, decisions that have to hold up when humans look closer.
